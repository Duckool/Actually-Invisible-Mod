# Actually Invisible

Ever made an invisibility potion expecting to be the stealthiest ninja on earth, just to have your dreams broken? Monsters still see you from unpredictable distances, players see your gear, and even if you strip, they'll still see your bubbles from far away.

### If you want to *actually* be invisible, then this is the mod for you!

![](link)

## THE GREAT 3 LAWS OF INVISIBILITY

*   **No Mob Detection:** While invisible, mobs won't target you unless you hit them. _(configurable)_
*   **No Material Presence:** While invisible, other players won't see any of your gear (armor or held items). You will always see your own gear. _(configurable)_
*   **No Eagle Farsight:** While invisible, other players won't see any bubble particles unless they are within 5 blocks. _(configurable)_

---

### CONFIGURATION
This mod has gamerules for you to configure it:

- `mob_can_detect_invisible` (default = false) // If mobs can target you while invisible.
- `mob_can_react_invisible` (default = true) // If mobs react to being hit while invisible.
- `is_gear_visible` (default = false) // If your gear is visible while invisible.
- `bubble_visible_range` (default = 5) // Distance in blocks others can see your bubbles (set 0 to disable).

---

### KNOWN ISSUES
Changing gamerules in-game doesn't apply immediately. This isn't a bug; the mod just needs a refresh (like re-applying the effect or re-equipping gear) to register the change. I'll fix this eventually.
